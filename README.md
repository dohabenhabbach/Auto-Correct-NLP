# Auto-Correct-NLP 

Auto-Correct-NLP 
<h1> Description </h1>
<ul>
  <li>This program suggests other words that are similar in usage to the input words based on a given text corpus. </li>
</ul>
<h1>Prerequisites</h1>
This program requires Python 3 and the following libraries: <br>
<ul>
  <li>Flask </li>
  </ul>
```
   pip install flask
```
<ul>
  <li>re </li>
  <li>Collection </li>
</ul>
```
   pip install Collection re
```
